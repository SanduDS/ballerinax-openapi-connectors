## Package overview
The `ballerinax/saps4hana.itcm.product` is a [Ballerina](https://ballerina.io/) connector for Product in SAPS4HANA Intelligent Trade Claims Management.

### Compatibility
|                    | Version                   |
|--------------------|---------------------------|
| Ballerina Language | Ballerina Swan Lake 2201.3.1|
| SAPS4HANA ITCM API | 1.0.0                     |
